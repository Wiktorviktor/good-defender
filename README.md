#  Good-Defender  (security for vpn)

**Good-Defender** - це мультиплатформенний агент для захисту кінцевих пристроїв від вторгнення, фішингу, шкідливого ПЗ і витоку даних.
Весь трафік примусово перенаправляється через VPN-тунель, а поза та без тунелю повністю блокується.

## основні можливості

- Повна ізоляція трафіку через VPN (OpenVPN,  або WireGuard).
- Локальний Firewall для блокування небезпечного трафіку.
- Централізована фільтрація на сервері.
- кросплатформена підтримка: Linux, Wsndows, Android.
- Антивірус, антифішинг, DNS-фільтрація на сервері.

  ## Компоненти
  **agent/** - клієнтська частина (Python)
  **server/** - VPN-сервер і фільтрація трафіку
  **docs/** - документація та архітектура проекту

  ## Мета
  Створити безпечне середовище для роботи в інтернеті, де все контролюється та захищяється, а клієнт немає можливості вийти за межі захищеного тунелю.
